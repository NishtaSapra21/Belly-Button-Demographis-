Belly Button Demographics

Human body is the source of thousands of bacteria. Still many bacteria are unknown and unstudied. This data analysis uses data of volunteers’ navel bacterial samples from across the country to make “Demographic Dashboard” and dynamic display charts using of Plotly, JavaScript, HTML and CSS.

Each volunteers who participated in this study given an unique ID. Volunteers and other researchers can see the “Belly Button” data using this dashboard and charts.

A dropdown menu will list the ID numbers of all the volunteers. When a volunteer ID is chosen from the dropdown menu, that person's demographics information, such as location, gender and age will be displayed.

Horizontal Bar Chart

This horizontal bar chart displays the top 10 bacterial species (OUTs) when an individual ID is selected from dropdown menu on webpage.

The horizontal bar chart displays the "sample_values", the "otu_ids" as the labels, and the "otu_labels" as the hover text for the bars on the chart.

Bubble Chart

This bubble chart displays the bacteria cultures per sample when an individual ID is selected from dropdown menu on webpage.

The bubble chart displays "otu_ids" as x-axis and marker colors, "sample_values" as y-axis and marker size, and "otu_labels" as hoover text values.

Gauge Chart

This gauge chart displays the weekly washing frequency's value and displays the value as a measure from 0-10 on the progress bar when an individual ID is selected from dropdown menu on webpage.
